# OpenAI Agents Deep Research

This is a TypeScript implementation of your phase-based deep-research workflow using the OpenAI Agents SDK.

## Install

```bash
npm install
cp .env.example .env
```

## Run

```bash
npm run dev
```

## What it does

- Builds a research plan
- Searches for diverse sources
- Fetches and analyzes selected pages
- Synthesizes validated claims and contradictions
- Produces a structured report with confidence scoring

## Main file

- `openai_agents_deep_research.ts`
